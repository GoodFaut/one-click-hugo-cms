Las páginas web como esta, utilizan un lenguaje de progtamación llamado HTML. Es el programa que hace que puedas visitar estos sitios web. Nuestro primer código es este:

```
<!DOCTYPE html>
<html>
<head>
<title>Título de la página</title>
</head>
<body>

<h1>Encabezado</h1>
<p>Párrafo</p>

</body>
</html>

```
